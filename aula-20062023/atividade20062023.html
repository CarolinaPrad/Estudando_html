Atividade - Tags de texto

Nesta atividade, você aprenderá a utilizar tags de texto para formatar o conteúdo em uma página web.

A seguir, temos um exemplo de trecho de código em HTML para enfatizar um texto:

<em> Texto enfatizado </em> -> Texto enfatizado

Experimente utilizar outras tags de texto, como <strong>, <u>, <sub>, <sup>, entre outras, para ver como elas alteram a aparência do texto 

        <Stong> Aqui temos uma frase em negrito </strong> -> Texto em negrito

        <i> Aqui uma frase em itálico </i> -> Texto em itálico
    
        <Small> Aqui uma com menor relevancia </Small> -> Texto com menor relevancia
    
        <u> Aqui uma frase sublinhada </u> -> Texto com sublinhado
    
        <Sup> Aqui um texto sobrescrita 22 = 4 </sup> -> Texto sobrescrito
    
        <sub> Aqui um texto Subscrita log24 = 2 </Sub> -> Texto subscrito
